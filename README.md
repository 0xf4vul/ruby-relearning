## 如何编译这本书？

这本书使用 Asciidoc 标准编写。编译这本书需要有 Ruby 的环境。

使用 `bundle install` 安装工具，并使用 `bundle exec rake build` 进行编译。

编译完成的电子书在 `./build` 目录下。

## 如何给这本书提供意见/修改？

本书使用 Git 进行维护，并使用 GitHub 对 Git 仓库进行管理。请使用 GitHub 的 Issue 和 Pull Request 系统为本书提供意见和修改。
